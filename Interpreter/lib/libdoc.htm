Scripting Windows Host libraries documentation

What are Scripting Windows Host libraries?

Scripting Windows Host libraries are extensions that allow you to do more things with it.
If you are experienced with Python, you should know that you need to import some libraries to make code extensions working. This is the case with Scripting Windows Host.
You can import them with import library[.cmd] resource parameters

Here's a list of some official libraries. Click one library name to read his documentation.

  • Windows.cmd
  • Printer.cmd
  • BIOS.cmd
  • Hardware.cmd
  • Math.cmd

    Windows.cmd

    This library displays Windows information and performs basic tasks.

    Example:

  • import Windows.cmd kernelversion variableKernelver
    Will save the Windows NT kernel version into the variable variableKernelver (You could call the result with %variableKernelver%)

    Printer.cmd

    This library displays and modified printer information Example:

  • import Printer.cmd default
    Will print the default printer into the screen
