Scripting Windows Host Console



Scripting Windows Host Console is a program that allows you to have more control on your system via command-line.
You can do many things, such as creating and deleting files and folders, ending processes or checking if a process is running. You can also encode and decode files in Base64, encrypt and decrypt text with a private key, run files and programs as administrator, reverse strings, view and change network settings and so on.
